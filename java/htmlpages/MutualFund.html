...about the visualization

This application shows the performance of a variety of mutual funds using data obtained from the web. A total of 15 parameters are associated with each fund in the source data and one additional value, sd, or the standard deviation of the performance, is calculated within the visual program.

...about the visual program

The 3 spatial axes correspond to the 1, 3, and 5 year performance; two additional parameters can be assigned to the size and color of the spheres. Furthermore, by clicking on a sphere with the mouse you can query any of the parameters assigned to that fund.

...about the conversion to web page

In the Mode menu, select Pick. Then click the mouse on a sphere. You'll see the Execution turn green as Data Explorer computes the result of your query.
