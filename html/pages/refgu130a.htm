Function

Convert an Object to a Screen Object

Syntax

output = ScreenObject ( object, position, display )

 

Inputs

Name

Type

Defaults

Description

object

field, group, or object

(none)

the object to turn into a screen object

position

 vector no default position where the object should be put

how

integer or string

0

coordinate system with reference to the position vector.
size scalar or vector 100 size of the screen object
depth integer or string 1 arrangement with other objects

Outputs

Name

Type

Description

screen

screen object

the reultant screen object

Functional Details

This module creates a screen object out of the input object. A screen object is different than a regular object. When rotating or zooming, a screen object does not change (it is not in the scene but in front of it.).

 

object
Specifies the object to change into a screen object.
 
position
is a 2-dimensional vector (or a 3-dimensional vector whose z-component is ignored) indicating the position of the screen object in the final image. In viewport-relative coordinates, [0 0] places the bar at the lower left, and [1 1] at the upper right. These same coordinates determine the reference point that is used to position the bar relative to its placement in the image (e.g., for position = [0 0], the lower left corner of the bar is placed in the lower left corner of the image).
 
how
the reference coordinate system for defining how position should be used. Valid values are viewport (0), pixel (1), world (2).
size
a scale used when rendering; for example when set to 200 the object will be rendered at twice the size.
depth
The display paramter controls where the Screen Object is displayed relative to all other Objects in the scene. Valid options are "behind", "equal", and "in front".
 

 Note: A screen object will not render in 3-d but only in 2-d.

Components

Creates a Screen Object containing the object.
